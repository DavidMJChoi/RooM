This is a toy project for personal learning purpose.

## Architecture
* **Core Components**:
  - **Router**: Lightweight HTTP router for request handling
  - **Markdown Processor**: Converts markdown to HTML with syntax highlighting
  - **Template Engine**: Go's built-in `html/template` with custom functions
  - **Static Assets**: Served directly from `/static` directory

* **Back end**: Go
* **Front end**: Go (`html/template`), Tailwind CSS
